OpenWhisk Documentation

OpenWhisk documentation is categrorized in three seperate sections based on the following three roles:

Users Developers Contributors

OpenWhisk Users:

Operators:

  • Compose: The easiest way to start using OpenWhisk locally is to use Docker Compose. GitHub repo incubator-openwhisk-devtools shows how to setup OpenWhisk with Docker Compose.
  • Kubernetes: incubator-openwhisk-deploy-kube has instructions on how to deploy OpenWhisk on a Kubernetes cluster.
  • Mesos: incubator-openwhisk-deploy-mesos can be used to deploy OpenWhisk to a Mesos cluster.
  • OpenShift: incubator-openwhisk-deploy-openshift repository contains the necessary templates and compatible docker images for deploying OpenWhisk on OpenShift.
  • Ansible: If you want to deploy OpenWhisk locally using Ansible, you first need to install Ansible on your development environment and follow instructions from README.md in main OpenWhisk repo.
  • Vagrant: A vagrant machine is also available to run OpenWhisk locally. incubator-openwhisk has a seperate section with instructions on deploying OpenWhisk using Vagrant.

End Users:

  • How to write an action?
  • OpenWhisk CLI: With Apache OpenWhisk you can easily create and run actions, or connect to other actions. Use the OpenWhisk Command Line Interface (CLI), target your Apache OpenWhisk instance, and run your first action in seconds.
  • OpenWhisk REST API After your OpenWhisk environment is enabled, you can use OpenWhisk with your web apps or mobile apps with REST API calls. For more details about the APIs for actions, activations, packages, rules, and triggers, see the OpenWhisk API documentation.
  • OpenWhisk API V1 Swagger: Here is the swagger file for OpenWhisk API V1.
  • Whisk Deploy: Whisk Deploy is an utility to help deploy all your OpenWhisk Packages, Actions, Triggers, and Rules using a single command.
  • API Gateway: OpenWhisk supports API Gateway based on OpenResty and Nginx.
  • OpenWhisk Catalog: OpenWhisk Catalog maintains the package catalog of openwhisk. OpenWhisk package catalog gives you an easy way to enhance your app with useful capabilities, and to access external services in the ecosystem.

OpenWhisk Developers

OpenWhisk Clients:

  • JavaScript: incubator-openwhisk-client-js is the JavaScript client library which Provides a wrapper around the OpenWhisk APIs.
  • incubator-openwhisk-client-go is a Go client library to access OpenWhisk API.
  • incubator-openwhisk-client-swift is a Swift based client SDK for OpenWhisk which can be used to connect to OpenWhisk. It partially implements the OpenWhisk REST API and allows you to invoke actions and fire triggers.
  • There is no official Python client for OpenWhisk at the moment. However, incubator-openwhisk-client-python shows an example of how OpenWhisk REST API can be used directly from Python.
