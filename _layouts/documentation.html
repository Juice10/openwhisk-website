Operators:

OpenWhisk supports many different types of deployment models. The easiest way to start deploying OpenWhisk is to get Docker installed on Mac, Windows or Linux. This does not give you a production deployment but gives you enough of the pieces to start writing functions and seeing them executing.

OpenWhisk Deployment using Docker-Compose

Another path to quickly start deploying OpenWhisk is to install it on a Kubernetes cluster. OpenWhisk can be installed on Minikube, or on a managed Kubernetes cluster provisioned from a public cloud provider, or on a Kubernetes cluster you manage yourself.

OpenWhisk Deployment using Kubernetes

A Vagrant machine is also available to run OpenWhisk on Mac, Windows PC or GNU/Linux. Downloading and install VirtualBox and Vagrant for your operating system and architecture.
